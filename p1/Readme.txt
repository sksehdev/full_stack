Project 1 for the Udacity Full Stack Nanodegree

The Project Contains three files 

Fresh_tomatoes.py: This file contains web client code which produces html file to render on the web browser when we use fresh_tomatoes.open_movies_page() function and pass a  list of movies objects.

media.py : This file contains movie class which is used to define the abstraction of movie object and defines the constructor for the object.

entertainment.py : The file contains three instances of movie class and calling of fresh_tomatoes.open_movies_page() function that is passed a list of movie objects.

The Project can seen by running the entertainment.py  file which will produce fresh_tomates.html  file and that will be rendered on the web page.
